# Personal Finance Tracker

A command-line finance tracker built with Python. It stores income and expenses in a JSON file, calculates balance, and generates category-based expense reports.

## Features

- Add income and expense transactions
- Save data locally in JSON
- Calculate current balance
- Generate expense report by category
- Unit test included

## How to Run

```bash
python app.py
```

## How to Test

```bash
python -m unittest
```

## Technologies

- Python
- JSON
- unittest
